Spacebase lets you visualize location-based sensor data on the fly

The Idea

Knowing when critical environmental-control systems are failing in a close environment is very important

Spacebase was conceived to provide a visualization of heat and light data in a closed environment, like a space shuttle

The Data

Spacebase uses the Intel Edison board with temperature and light sensors for real-time data collection

However, in the future, Spacebase will also accept arbitrary location-based data for visualization

The Application

The Spacebase application is built in Python Flask and AngularJS

Spacebase is currently deployed to the Microsoft Azure Cloud to ensure high-availability

How It Looks

Our current visualization is a heatmap that plots the intensity of data readings
